Lista de tarefas que salva em arquivo em Python

Projeto simples de uma lista de tarefas desenvolvido em Python, com foco em prática de lógica de programação e manipulação de arquivos.

Funcionalidades:
- Adicionar tarefas
- Listar tarefas
- Marcar como concluída
- Desmarcar tarefa
- Editar tarefas
- Remover tarefas
- Buscar tarefas por nome
- Salvamento automático em arquivo ".txt"

Armazenamento:
-As tarefas são armazenadas em um arquivo `tarefas.txt`, simulando um banco de dados simples.

Tecnologias utilizadas:
- Python
- Manipulação de arquivos (leitura e escrita)
- Estruturas de dados (listas e dicionários)

Objetivo:
Praticar conceitos fundamentais de programação, como:
- Estruturas condicionais
- Laços de repetição
- Funções
- Organização de código

O arquivo tarefas.txt é criado automaticamente ao executar o programa.
